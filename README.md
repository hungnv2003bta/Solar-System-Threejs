HOW TO RUN Solar-System-Threejs

1. Clone the repository:
   ```bash
   git clone https://github.com/hungnv2003bta/Solar-System-Threejs.git
2. Open "Solar-System-Threejs" folder have just cloned
  Note:  If you created an empty folder and cloned the repository into it, please close the app and reopen only the "Solar-System-Threejs" folder.
3. Run the `index.html` file with Live Server:
  . If you are using Visual Studio Code (VS Code), you can use the Live Server extension:
      . Right-click on the index.html file.
      . Select "Open with Live Server".
